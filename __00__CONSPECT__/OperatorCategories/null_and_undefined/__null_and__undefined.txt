В JavaScript для проверки переменной на равенство `null` 
можно использовать оператор строгого равенства `===`, 
который проверяет как значение, так и тип:

```javascript
let variable = null;

if (variable === null) {
    console.log('Переменная равна null');
} else {
    console.log('Переменная не равна null');
}
```

Если нужно проверить переменную 
на `null` или `undefined`, 
можно использовать оператор 
нестрогого равенства `==`, 
который приведет оба значения к одному типу:

```javascript
if (variable == null) {
    console.log('Переменная равна null или undefined');
}
```

Однако, использование строгого равенства 
считается лучшей практикой, так как оно 
предотвращает неожиданные 
результаты из-за неявного приведения типов.
